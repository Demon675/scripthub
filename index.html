Teile und entdecke Roblox Scripts

Die beste Plattform für hochwertige Roblox-Scripts. Lade deine eigenen Kreationen hoch oder finde nützliche Scripts für deine Lieblingsspiele.

Dokumentation & Anleitung

Schnelleinstieg

  1. Script suchen: Scrolle durch die Script-Liste oder nutze die Kategorien.
  2. Script anzeigen & kopieren: Klicke auf Script anzeigen bei einem Script. Im Modal kannst du den Lua-Code kopieren.
  3. Eigenes Script hochladen: Klicke auf Jetzt Script hochladen oder gehe zum Upload-Bereich. Fülle das Formular aus und lade dein Script hoch.

Feature-Übersicht

  • Script-Upload: Scripts werden über das Upload-Formular direkt im Browser gespeichert (localStorage).
  • Script-Liste: Alle Scripts werden als Karten angezeigt, inkl. Kategorie, Tags, Beschreibung und Bewertung.
  • Script anzeigen Modal: Beim Klick auf Script anzeigen öffnet sich ein Modal mit dem vollständigen Lua-Code und Kopierfunktion.
  • localStorage: Scripts werden im Browser gespeichert und bleiben auch nach einem Neuladen erhalten.
  • Kategorien & Tags: Scripts können nach Kategorie und Tags gefiltert werden (Feature kann erweitert werden).
  • Unendliches Scrollen: Beim Scrollen werden automatisch mehr Scripts geladen.

FAQ & Hinweise

  • Wo werden meine Scripts gespeichert? Im localStorage deines Browsers. Sie sind nur auf deinem Gerät sichtbar.
  • Kann ich Scripts löschen? Aktuell nicht direkt, aber du kannst den localStorage im Browser leeren.
  • Wie kann ich Scripts teilen? Kopiere den Lua-Code und teile ihn manuell.
  • Wie kann ich Feedback geben? Schreibe uns über das Kontaktformular auf der Hauptseite.
